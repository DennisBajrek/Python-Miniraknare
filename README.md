# Python-Miniraknare
A simple calculator in Python. The calculator solves equations in addition, subtraction, multiplication and division.

The code follows the PEP 8 (Python style Guide) to make sure that the code is clean and readable. 
PEP 8 guidelines are followed by using:
- Clear and descriptive variable and function names written in snake_case.
- Consistent 4-space indentation.
- Proper use of blank lines to separate logical sections.
- Comments that explain why certain decisions are made, improving readability.
